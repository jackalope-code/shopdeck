# Running development mode
1. `git clone https://github.com/jackalope-code/shopdeck.git`
2. `cd shopdeck`
3. `npm install`
4. `node backend/server.js` Start the server
5. ` npm start` Start Nextjs

# Running a build
1. `git clone https://github.com/jackalope-code/shopdeck.git`
2. `cd shopdeck`
3. `npm install`
4. `npm build`
5. `node backend/server.js` Start the server
6. `npm start` Start Nextjs

Note that the user.json file contains sensitive data! (OAuth tokens)

Modeled after SitDeck.
